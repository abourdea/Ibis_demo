# Ibis Demo

## Summary

This repository is a demo space dedicated to describing Git, GitHub, and GitKraken (how the tools work together, use cases for each, and how the infrastructure created by these tools can greatly improve efficiency, reproducibility, and learning).

## Git

[Git](https://git-scm.com/) is a free, open-source version-control software.

## GitHub

[GitHub](https://github.com/) is a cloud-based version-control platform that stores tracked code repositories.

## GitKraken

[GitKraken](https://www.gitkraken.com/) is a graphical user interface (GUI) tool for tracking changes to local and remote repositories with GitHub.
